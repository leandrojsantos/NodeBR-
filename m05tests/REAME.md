#### 🎲 Rodando o Projeto

```bash
# Clone este repositório
$ git clone git@github.com:leandrojsantos/node-br.git

# Acesse a pasta do projeto no terminal/cmd, vá para a pasta raiz do projeto
$ cd node-br/m05tests

# Instale as dependências
$ npm install

# para iniciar o projeto
$ npm test

```
